# Catch a condition

This is a small wrapper around
[`tryCatch()`](https://rdrr.io/r/base/conditions.html) that captures any
condition signalled while evaluating its argument. It is useful for
situations where you expect a specific condition to be signalled, for
debugging, and for unit testing.

## Usage

``` r
catch_cnd(expr, classes = "condition")
```

## Arguments

- expr:

  Expression to be evaluated with a catching condition handler.

- classes:

  A character vector of condition classes to catch. By default, catches
  all conditions.

## Value

A condition if any was signalled, `NULL` otherwise.
