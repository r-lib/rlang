As of rlang 1.0.0, `with_handlers()` is deprecated. Use the base
functions or the experimental
[`try_fetch()`](https://rlang.r-lib.org/dev/reference/try_fetch.md)
function instead.

## Usage

``` r
with_handlers(.expr, ...)

calling(handler)
```
