# as_function() has nice errors

      (expect_error(as_function(1)))
    Output
      <error/rlang_error>
      Error: Can't convert `1`, a double vector, to a function.
    Code
      (expect_error(as_function(1, arg = "foo")))

      <error/rlang_error>
      Error: Can't convert `foo`, a double vector, to a function.
